G-Tab (Grocery Tablet)

Grocery shopping for the best produce but at the cheapest price can be a big hassle for busy individuals. The G-Tab is a device that allows users to customize the tablet to order their necessary groceries from their favorite groceries at the cheapest price with the push of just a few buttons. This device was built using a Raspberry to allow for customization of objects to order and a special 'Sale' button. This button allows users to find the cheapest good within a 10 mile radius. Customized grocery items, vendors, and prices were web-scraped using Python and stored in a Mongo database. Upon ordering groceries, these items would be fetched to provide users with the best total possible.
